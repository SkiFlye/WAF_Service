Настройки

API Ключ
Внимание! API-ключ нельзя изменить или перегенерировать. Сохраните его в надежном месте. При утере ключа потребуется создать новый аккаунт.
Rate Limit
При превышении лимита IP блокируется на 60 секунд
Инструкция по подключению
Шаг 1: Скачайте файл декоратора
Скачать waf_check.py

Скачайте файл и поместите его в корень вашего Flask проекта

Шаг 2: Импортируйте и используйте декоратор в вашем Flask приложении 
from waf_check import waf_check

# Настройки WAF
WAF_URL = "http://{{ request.host.split(':')[0] }}:8080"
API_KEY = "{{ user.api_key }}"

# Пример использования на маршруте
@app.route('/login', methods=['GET', 'POST'])
@waf_check(WAF_URL, API_KEY)
def login():
    # Ваш код обработки логина
    pass

# Для всех маршрутов, которые нужно защитить, добавьте декоратор:
@app.route('/register', methods=['GET', 'POST'])
@waf_check(WAF_URL, API_KEY)
def register():
    pass

@app.route('/user/', methods=['GET', 'PUT', 'DELETE'])
@waf_check(WAF_URL, API_KEY)
def user(user_id):
    pass
Шаг 3: Запустите ваше приложение
Готово! Теперь все запросы к защищенным маршрутам будут проходить через WAF.
Удаление аккаунта

Удаление аккаунта приведет к потере всей статистики и настроек. API-ключ станет недействительным.
