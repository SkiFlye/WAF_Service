ПОЯСНИТЕЛЬНАЯ ЗАПИСКА


WAF Service — Web Application Firewall
1. Название проекта
WAF Service

2. Автор проекта
Никифоров Матвей

3. Описание идеи
WAF Service — облачный сервис для защиты веб-сайтов от хакерских атак. Сервис работает как обратный прокси: принимает запросы к сайту, анализирует их и блокирует атаки (SQL инъекции, XSS, Path Traversal). Каждый владелец сайта получает API-ключ и личный кабинет со статистикой.

4. Описание реализации
Основные классы:
User : модель пользователя (email, пароль, API-ключ, настройки)
Statistic : модель логов запросов (тип, метод, IP, URL, правило)
BlockedIP : модель заблокированных IP-адресов
WAFProxy : асинхронный прокси-сервер на aiohttp
WAFCore : ядро WAF (проверка правил, rate limiting)
RateLimiter : ограничитель частоты запросов с одного IP

Реализованные системы:
Регистрация и авторизация : создание аккаунта, вход по email и паролю, хеширование паролей
API-ключи : генерация уникального ключа при регистрации, идентификация пользователя по ключу
Правила WAF : 9 встроенных правил (SQLi, XSS, Path Traversal, Command Injection) с уровнями серьезности
Проверка атак : анализ URL, параметров запроса и заголовков по регулярным выражениям
Rate limiting : ограничение запросов с одного IP, настраиваемый лимит (10-1000 в минуту)
Блокировка IP : автоматическая (при превышении лимита) и ручная (через интерфейс)
Личный кабинет : дашборд с графиком атак, таблица последних атак, управление правилами
Декоратор @waf_check : для интеграции WAF в сторонние Flask-приложения

5. Технологии
Python 3.14 : язык программирования
Flask : веб-фреймворк для сайта и личного кабинета
aiohttp : асинхронный прокси-сервер
SQLAlchemy : ORM для работы с базой данных
SQLite : база данных
Flask-Login : аутентификация пользователей
Waitress : production WSGI сервер
Bootstrap : CSS-фреймворк
Chart.js : построение графиков

6. Функционал
* Регистрация и вход пользователей
* Генерация API-ключей
* 9 правил обнаружения атак
* Rate limiting с настраиваемым лимитом
* Автоматическая блокировка IP
* Ручная блокировка IP
* Дашборд с графиком атак
* Таблица последних атак с сортировкой
* Управление правилами (вкл/выкл)
* Копирование API-ключа
* Декоратор для интеграции в Flask
* Поддержка GET, POST, PUT, DELETE
* Проксирование чистых запросов

7. Особенности реализации
Асинхронный прокси : высокая производительность благодаря aiohttp
Многопользовательская архитектура : данные каждого пользователя изолированы
Адаптивный интерфейс : работает на всех устройствах
